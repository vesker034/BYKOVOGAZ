{% trans "История открытия" %}

{% blocktrans trimmed %}Южно-Кисловское газоконденсатное месторождение выявлено в результате сейсморазведочных исследований, выполненных в 1966 году. По результатам работ была подготовлена структура, установленная поисково-разведочной скважиной №14 в 1974 году.{% endblocktrans %}

{% blocktrans trimmed %}В ходе этой доразведки было выявлено более 50 перспективных локальных структур и скоплений углеводородов. Наиболее значимой стала зона в пределах Южно-Кисловской структуры, где подтверждены промышленные запасы газа и конденсата.{% endblocktrans %}

{% trans "Переоценка запасов 2006 года" %}

{% blocktrans trimmed %}В 2006 г. была проведена независимая оценка запасов, потребовавшая обновления проектных решений и уточнения геологической модели. По результатам работ были скорректированы показатели по запасам газа и конденсата Южно-Кисловского месторождения.{% endblocktrans %}

{% blocktrans trimmed %}Полученные результаты создали технологическую и инвестиционную основу для дальнейшего освоения участка.{% endblocktrans %}

{% trans "Утвержденные запасы" %}

{% blocktrans trimmed %}В 2006 году в рамках ГКЗ России были утверждены запасы углеводородов Южно-Кисловского газоконденсатного месторождения по категориям С1.{% endblocktrans %}

{% trans "Опытно-промышленная эксплуатация" %}

{% blocktrans trimmed %}Для ввода Южно-Кисловского газоконденсатного месторождения в опытно-промышленную эксплуатацию компанией «Инвестпроект» в 2005 г. разработан проект опытно-промышленной эксплуатации.{% endblocktrans %}

{% blocktrans trimmed %}Для уточнения геологического строения месторождения компанией была также выполнена сейсморазведка МОГТ-3D общим объемом 63 км2.{% endblocktrans %}
